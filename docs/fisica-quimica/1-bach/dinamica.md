# Dinámica

Recursos para trabajar las leyes de Newton y su aplicación a la resolución de problemas.

## Contenidos

- Fuerzas y diagramas de cuerpo libre
- Leyes de Newton
- Fuerza de rozamiento
- Planos inclinados
- Tensión
- Movimiento circular
- Sistemas de cuerpos enlazados
