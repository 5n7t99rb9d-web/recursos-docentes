# Cinemática

Recursos para estudiar la descripción matemática del movimiento y resolver problemas de cinemática en una y dos dimensiones.

## Contenidos

-Magnitudes cinemáticas: posición, desplazamiento, velocidad y aceleración
-Movimiento rectilíneo uniforme
-Movimiento rectilíneo uniformemente acelerado
-Caída libre y lanzamiento vertical
-Movimiento circular uniforme
-Composición de movimientos
-Lanzamiento horizontal y movimiento parabólico
